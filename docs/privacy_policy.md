# 개인정보처리방침

**최종 수정일**: 2026년 5월 9일
**적용 대상**: BabyNote (이하 "서비스")

BabyNote는 사용자의 개인정보 보호를 매우 중요하게 생각하며, 「개인정보 보호법」 등 관련 법령을 준수합니다. 본 방침은 서비스가 어떤 정보를 수집·이용하는지, 어떻게 보호하는지를 설명합니다.

---

## 1. 수집하는 개인정보

### 회원가입 / 인증
- 이메일 주소, 비밀번호(해시 저장), 표시 이름

### 자녀 정보 (사용자가 직접 입력)
- 이름, 생년월일, 성별

### 기록 데이터
- 수유 / 수면 / 기저귀 / 성장 측정 / 분유·기저귀 재고
- 단골 병원, 예방접종 일정
- 사용자가 첨부한 사진 (이유식 등)

### 자동 수집
- 디바이스 식별자(앱 로그인 세션 유지용)
- 크래시 로그(에러 모니터링용)

**위치 정보, 연락처, 카메라(이유식 사진 외) 등은 수집하지 않습니다.**

---

## 2. 수집·이용 목적

| 목적 | 사용되는 정보 |
|---|---|
| 회원 식별 / 로그인 | 이메일, 비밀번호 |
| 핵심 기록 서비스 제공 | 자녀 정보, 기록 데이터 |
| 가족 공유 | 회원 식별자, 자녀 ID |
| 통계 / 차트 제공 | 기록 데이터 |
| 알림 발송 (분유 부족 등) | 기록 데이터 |
| 오류 진단 | 크래시 로그 |
| 결제 처리 | 결제 토큰(RevenueCat) |

---

## 3. 보유·이용 기간

회원 탈퇴 시 즉시 삭제합니다. 다만 다음의 경우 관계 법령에 따라 일정 기간 보관할 수 있습니다.

- 결제 기록: 5년 (전자상거래법)
- 부정 이용 기록: 1년 (보안)

---

## 4. 제3자 제공

**제3자에게 개인정보를 제공하지 않습니다.** 다음의 처리 위탁만 진행합니다.

| 수탁자 | 위탁 업무 | 처리 항목 |
|---|---|---|
| Supabase Inc. (미국) | 데이터베이스 / 인증 호스팅 | 회원/자녀/기록 데이터 |
| Sentry (미국) | 에러 모니터링 | 크래시 로그(개인정보 X) |
| Google Maps Platform | 병원 자동완성 | 검색어(이름) |
| RevenueCat (미국) | 결제 검증 | 결제 토큰 |

위 수탁자들은 각자의 개인정보보호 정책을 따르며, 본 서비스의 데이터 보안 요구사항을 준수합니다.

---

## 5. 사용자의 권리

언제든지 다음 권리를 행사하실 수 있습니다.

- **조회**: 앱 내 설정 → 데이터 내보내기(CSV)
- **수정/삭제**: 각 기록의 편집/삭제 기능
- **탈퇴**: 설정 → 계정 → 회원 탈퇴 (모든 데이터 즉시 삭제)
- **문의**: privacy@babynote.app

---

## 6. 보안 조치

- 모든 통신 HTTPS/TLS 1.2+ 암호화
- Postgres Row Level Security(RLS)로 사용자별 데이터 격리
- 비밀번호 bcrypt 해시 저장
- 결제 정보는 직접 보관하지 않음 (Apple / Google / RevenueCat 처리)

---

## 7. 만 14세 미만 아동의 개인정보

이 앱은 부모/보호자가 자녀의 양육 정보를 기록하기 위한 도구이며, **자녀가 직접 회원가입하지 않습니다.** 부모/보호자가 입력하는 자녀 정보는 본 방침에 따라 보호됩니다.
