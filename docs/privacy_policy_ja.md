# プライバシーポリシー

**最終更新日**: 2026年5月9日
**対象サービス**: BabyNote (以下「本サービス」)

BabyNote はお客様のプライバシー保護を重要視し、個人情報の保護に関する法律
(日本)、GDPR (EU)、その他適用される法令を遵守します。本ポリシーは、本サービスが
どのような情報を収集し、どのように利用・保護するかを定めるものです。

---

## 1. 収集する情報

### アカウント / 認証
- メールアドレス、パスワード (ハッシュ化保存)、表示名

### お子様の情報 (ユーザー自身が入力)
- 名前、生年月日、性別

### 記録データ
- 授乳 / 睡眠 / おむつ / 成長記録、ミルク・おむつ在庫
- かかりつけ病院、予防接種スケジュール
- ユーザーが添付した写真 (離乳食、発疹、けが等)

### 自動収集
- デバイス識別子 (ログイン状態の維持に使用)
- クラッシュログ (エラー監視)

**位置情報、連絡先、明示的に許可した写真添付以外のカメラ機能は収集しません。**

---

## 2. 利用目的

| 目的 | 利用する情報 |
|---|---|
| 会員識別・ログイン | メール、パスワード |
| 中核機能の記録サービス提供 | お子様情報、記録データ |
| 家族共有 | 会員識別子、子供 ID |
| 統計・グラフ提供 | 記録データ |
| 通知発信 (ミルク不足、予防接種等) | 記録データ |
| エラー診断 | クラッシュログ |
| 決済処理 | 決済トークン (RevenueCat) |

---

## 3. 保有期間

会員退会時に即時削除します。ただし、関連法令により以下の例外があります。

- 決済記録: 5年 (電子商取引関連法)
- 不正利用防止記録: 1年 (セキュリティ)

EU 在住のユーザーには GDPR 第17条 (削除権) に基づき、法的保管義務がない限り
30日以内に削除します。

---

## 4. 第三者提供

**個人情報を第三者に販売することはありません。** 以下の業務委託のみ行います。

| 委託先 | 業務 | 提供データ |
|---|---|---|
| Supabase Inc. (米国) | データベース・認証ホスティング | 会員・子供・記録データ |
| Sentry (米国) | エラー監視 | クラッシュログ (個人情報なし) |
| Google Maps Platform | 病院検索の自動補完 | 検索クエリ |
| RevenueCat (米国) | 決済検証 | 決済トークン |

各委託先は各社のプライバシーポリシーおよび当社のデータ保護要件を遵守します。

---

## 5. ユーザーの権利

いつでも以下の権利を行使できます。

- **照会**: アプリ内 [設定 → データエクスポート (CSV)]
- **修正・削除**: 各記録の編集・削除機能
- **退会**: [設定 → アカウント → 退会] (全データ即時削除)
- **問い合わせ**: support.babynote@gmail.com

EU 在住のユーザーは、お住まいの地域のデータ保護機関に苦情を申し立てる権利が
あります。

---

## 6. セキュリティ対策

- すべての通信を HTTPS/TLS 1.2+ で暗号化
- Postgres Row Level Security (RLS) によりユーザーごとのデータ分離
- パスワードは bcrypt によりハッシュ化保存
- 決済情報は Apple / Google / RevenueCat が処理し、当社では保管しません

---

## 7. お子様の個人情報

本アプリは保護者がお子様の養育情報を記録するためのツールであり、**お子様自身は
会員登録しません**。保護者が入力するお子様情報は本ポリシーに従って保護されます。

日本の個人情報保護法 (満14歳未満は法定代理人の同意)、米国 COPPA (13歳未満)、
EU GDPR-K (16歳未満) を遵守します。

---

## 8. ポリシー変更のお知らせ

本ポリシーが変更される場合、アプリ内通知またはメールにより事前に案内します。
